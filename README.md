# Unlearning Demo (Dummy)

### Prerequisite
- Python

### Setup
- Git Clone: `git clone https://github.com/devansh-srivastav/esd_unlearning_demo_dummy`
- Install Requirements : `pip install -r requirements.txt`

### Usage
`streamlit run app.py`

As you change the code, you can see the changes in your localhost.

### Deployment

- Push the code (app.py and requirements.txt) to this Github Repository
- Any change you make and push to GitHub will be automatically updated in the deployment.
